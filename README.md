# WakuWakuDramaClub
SS集／5ch反應集，這類影片創作者花很多時間剪輯，但剪輯通常只是枯燥的TTS/上字幕的反覆作業，創作過程中最有價值的部份是寫劇本，WakuWakuDramaClub能讓創作者寫完劇本，跳過剪輯，直接渲染影片，讓創作者專注在選題和創意寫作，而不是繁瑣作業。

支援模板系統，不會畫圖也能用別人分享的素材，包含角色、背景、音效。透過資料夾檔案結構定義模板，拖曳資料夾就能匯入模板。

# 工作流比較：傳統 V.S WakuWakuDramaClub

## 傳統
```text
想劇本
	↓
生成 TTS
	↓
匯入剪輯軟體
	↓
排字幕
	↓
角色動畫
	↓
背景切換
	↓
調 Timeline
	↓
Render
```

## WakuWakuDramaClub
```text
寫很接近自然語言的劇本
	↓
Render
```
