# 劇本語法

劇本語法的目標是讓創作者用接近劇本的方式描述影片，而不是編輯 timeline。

每一行代表一個事件或演出指令：

```text
背景 default/文藝社團 早上
default/毛豆 登場 在 左
default/毛豆 「你好。」 表情 高興
default/毛豆 移動 到 右 在 0.5
等待 1.0
音效 default/問號
```

## 資源 ID

主要資源使用 `pack/name`：

```text
default/毛豆
default/文藝社團 早上
default/問號
```

角色的表情與動作是角色自己的子資源，只寫子資源名稱：

```text
default/毛豆 「你好。」 表情 高興 動作 mic
```

## 對話

角色對話可以使用 shorthand：

```text
default/毛豆 「你好。」
```

也可以使用正式指令：

```text
對話 default/毛豆 「你好。」
```

對話文字使用全形引號 `「」`。

對話可以加上表情與動作：

```text
default/毛豆 「今天也要努力。」 表情 高興
default/毛豆 「交給我吧。」 表情 determined 動作 mic
```

語法：

```text
actor 「dialogue」 [表情 expression] [動作 pose]
對話 actor 「dialogue」 [表情 expression] [動作 pose]
```

## 背景

切換背景：

```text
背景 default/文藝社團_早上
```

語法：

```text
背景 background
```

## 音效

播放音效：

```text
音效 default/問號
```

語法：

```text
音效 audio
```

## 等待

等待指定秒數：

```text
等待 1.0
```

語法：

```text
等待 seconds
```

## 登場

角色登場：

```text
default/毛豆 登場
```

指定登場位置：

```text
default/毛豆 登場 在 左
default/毛豆 登場 在 中
default/毛豆 登場 在 右
```

正式語法：

```text
登場 default/毛豆
登場 default/毛豆 在 左
```

語法：

```text
actor 登場 [在 anchor]
登場 actor [在 anchor]
```

`在` 是可選 option。沒有指定時，使用預設登場位置。

## 移動

角色移動到指定位置：

```text
default/毛豆 移動 到 右
```

指定起點與終點：

```text
default/毛豆 移動 從 左 到 右
```

指定移動時間：

```text
default/毛豆 移動 到 右 在 0.5
default/毛豆 移動 從 左 到 右 在 0.5
```

正式語法：

```text
移動 default/毛豆 到 右
移動 default/毛豆 從 左 到 右
移動 default/毛豆 從 左 到 右 在 0.5
```

語法：

```text
actor 移動 [從 anchor] 到 anchor [在 seconds]
移動 actor [從 anchor] 到 anchor [在 seconds]
```

`到` 是必要 option。`從` 是可選 option；沒有指定時，角色會從目前位置移動到目標位置。`在` 是可選 option，用來指定移動秒數。

Option statement 可以任意順序出現：

```text
default/毛豆 移動 從 左 到 右
default/毛豆 移動 到 右 從 左
```

兩者語意相同。

但 primary statement 必須先出現；以下不是合法語法：

```text
從 左 移動 default/毛豆 到 右
```

## 站位

目前支援的站位：

```text
左
中
右
```

站位可用於：

```text
登場 ... 在 anchor
移動 ... 從 anchor
移動 ... 到 anchor
```

## 範例

```text
背景 default/文藝社團 早上
音效 default/問號

default/毛豆 登場 在 左
default/毛豆 「這裡就是新的劇場嗎？」 表情 高興

default/潔莉諾 登場 在 右
default/潔莉諾 「沒錯，這裡由我來管理。」 表情 determined

default/毛豆 移動 從 左 到 中 在 0.5
default/毛豆 「那我站中間好了。」 表情 發呆

等待 1.0
背景 default/文藝社團 晚上
default/潔莉諾 「燈怎麼突然關了？」 表情 nervous
```

## 指令總覽

```text
背景 background
音效 audio
等待 seconds

對話 actor 「dialogue」 [表情 expression] [動作 pose]
actor 「dialogue」 [表情 expression] [動作 pose]

登場 actor [在 anchor]
actor 登場 [在 anchor]

移動 actor [從 anchor] 到 anchor [在 seconds]
actor 移動 [從 anchor] 到 anchor [在 seconds]
```

## 目前注意事項

這份文件描述目標語法。若目前實作尚未完全支援某些 option，例如 `登場 在` 或 `移動 從`，應以本文件作為後續實作方向。
